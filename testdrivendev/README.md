TDD 학습
===============

# 소개

- Test-Driven Development:By Example 책을 읽고 실습 및 내용을 정리한다.

## 목표

- TDD의 목표는 깔끔한 코드를 얻는 것
- 한번에 목표를 도달하기 어려우니 나누워서 정복하자.


## 1부 정리

- 테스트 주도개발의 리듬
  1. 재빨리 테스트를 하나 추가한다.
  2. 모든 테스트를 실행하고 새로 추가한 것이 실패하는지 확인한다.
  3. 코드를 조금 바꾼다.
  4. 모든 테스트를 실행하고 전부 성공하는지 확인한다.
  5. 리팩토링을 통해 중복을 제거한다.
